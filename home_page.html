How to play:

2048 is a game in which tiles with numbers appear on the board. Use the keyboard arrows to move the tiles around the board. Pressing Up will move all tiles upwards. Pressing Right will move all tiles to the right side of the board, and so on. Each time you make a move, a new tile will appear. The aim of the game is to make tiles with the same value bump into one another. When two tiles with the same value collide, they merge into a single tile with double the value. So if two tiles with the value of “2” collide, they turn into a single tile with the value of 4. If two tiles with the value of 4 collide, they merge into a single tile with the value of 8, and so on. You win when you manage to create one tile with the value of 2048. Good luck!
